URBAN EXPANSION DETECTION SYSTEM

Help and Instructions

Aims
This website aims to make SDG Indices more accsessible, which involves the information of population, Land Use Efficiency(LUE), Abstract Achieved Population Density in Expansion Areas(AAPDEA) to provide comparative analysis at different levels.

Function Toolbar
Tool Function Description
Basemap Switcher: Used for changing different base maps.
Bookmarks: Users can save the current view as spatial bookmarks for revisiting
Layer Groups: Differnt layers of LULC Maps, satellite images and spatial boundaries are provided in this page. Layer opacity can also be configured.
Temporal Slider: Different periods of LULC Maps and Built-up areas are shown in this page, while users can access this information through an interactive slide bar.
Statistics: - Population: Population data is obtained from the Civil Affairs Bureau of the Kaohsiung City Government, which is investigated based on administrative districts level.
- SDG Indices: Two SDG indices are implemented in this visualization tool.
  1. Abstract Achieved Population Density in Expansion Areas (AAPDEA)

    $$ AAPEDA= {Pop_{t+n}-Pop_{t}/Urb_{t+n}-Urb_{t}} $$

  2. Land Use Efficiency (LUE) is the ratio of the land consumption rate (LCR) to the population growth rate (PGR).

    $$LCR= {ln(Urb_{t+n})-ln(Urb_{t}) \over Δt}$$

    $$PGR= {ln(Pop_{t+n})-ln(Pop_{t}) \over Δt}$$

    $$LUE= {LCR \over PGR}$$

  3. Urban Expansion: Shannon Entropy(H) is calculated to represent urban expansion patterns, by measuring the changing balance between each land cover.

    $${H=-\sum _{i}p_{i}\log _{b}p_{i}}$$
Clear All Layers: All added layers can be clear through this button.
Print Map: The current map view can be print by clicking this button.
